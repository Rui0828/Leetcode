# 3666. Minimum Operations to Equalize Binary String
> 題目：https://leetcode.com/problems/minimum-operations-to-equalize-binary-string/description/  

## 1. 位置不重要，只有數量重要  

每次操作可以自由選擇 $k$ 個位置翻轉。因此我們不需要追蹤哪些位置是 `0` 或 `1`，只需要關心 `0` 的總數量。  

設 $n = \lvert s \rvert$， $z$ 為 `0` 的個數， $w = n - z$ 為 `1` 的個數， $b = n - k$ 代表每次操作中沒有被翻轉的位置數量。  

每次操作從 $k$ 個被選位置中，選了 $a$ 個 `0`（翻成 `1`）和 $k - a$ 個 `1`（翻成 `0`）。`0` 的數量變化為：  

$$z' = z - a + (k - a) = z + k - 2a$$  

目標是從 $z$ 經過若干步到達 $0$。  


## 2. 操作次數分配
**觀察：**  
進行 $m$ 次操作，每個位置被翻轉了若干次。被翻轉偶數次的位置回到原始狀態；被翻轉奇數次  
的位置改變狀態。因此：  
- 原本為 `0` 的位置需被翻轉奇數次。  
- 原本為 `1` 的位置需被翻轉偶數次（含0 次）。  

設每個位置 $j$ 的翻轉次數為 $t_j \in \{0, 1, \dots, m\}$。  

**限制：**  

1. $\sum_{j=1}^{n} t_j = mk$，且 $t_j \in \{0, 1, \dots, m\}$。（每次操作翻轉 $k$ 個位置，共操作 $m$ 次）。  
2. 原本為 `0` 的位置： $t_j$ 為奇數（至少 1）。  
3. 原本為 `1` 的位置： $t_j$ 為偶數（含 0）。  

因為 $t_j$ 不能任意分配，每次是選固定 $k$ 個來操作，因此需要能把 $\{t_j\}$ 拆解成 $m$ 個操作集合 (即第 1. 項限制： $\sum_{j=1}^{n} t_j = mk$)。  
> 直覺上：只要總量對了、每個位置不動超過 $m$ 次、奇偶性正確，就會存在合法方案。  


因此問題簡化為：找最小的 $m$，使得存在非負整數 $t_1, \dots, t_n$ 滿足以上限制。  


## 3. 推導三個條件

設 $z$ 個 `0` 位置的翻轉次數為 $f_1, \dots, f_z$（皆為奇數， $\ge 1$， $\le m$）， $w$ 個 `1` 位置的翻轉次數為 $g_1, \dots, g_w$（皆為偶數， $\ge 0$， $\le m$）。  


### 條件 (a) 討論每個位置的翻轉次數的最小值

$$\sum f_i \ge z \cdot 1 = z, \qquad \sum g_i \ge 0$$

故 $mk = \sum f_i + \sum g_i \ge z$，即：  

$$\boxed{mk \ge z \quad\Longleftrightarrow\quad m \ge \left\lceil \frac{z}{k} \right\rceil}$$

**小結：** 至少需要 $\lceil z/k \rceil$ 次操作。  

### 條件 (b) 討論 $m$ 奇偶情況的限制

**當 $m$ 為奇數時：**  

- `0` 位置需要 $t_j$ 為奇數， $t_j \le m$。因為 $m$ 本身為奇數，最大值為 $m$。  
- `1` 位置需要 $t_j$ 為偶數， $t_j \le m$。因為 $m$ 為奇數，最大的偶數為 $m - 1$。  

$$\sum t_j \le z \cdot m + w \cdot (m-1) = nm - w$$  

要求 $mk \le nm - w$，即 $w \le m(n-k) = mb$，即：  

$$\boxed{m \ge \left\lceil \frac{w}{b} \right\rceil \quad (m \text{ 為奇數})}$$

**當 $m$ 為偶數時：**  

- `0` 位置需要 $t_j$ 為奇數， $t_j \le m$。因為 $m$ 為偶數，最大的奇數為 $m - 1$。  
- `1` 位置需要 $t_j$ 為偶數， $t_j \le m$。因為 $m$ 本身為偶數，最大值為 $m$。  

$$\sum t_j \le z \cdot (m-1) + w \cdot m = nm - z$$  

要求 $mk \le nm - z$，即 $z \le mb$，即：  

$$\boxed{m \ge \left\lceil \frac{z}{b} \right\rceil \quad (m \text{ 為偶數})}$$  

**備註：** $m$ 為奇數時上界限制用 $w$， $m$ 為偶數時用 $z$。此不對稱性來自 `0` 需要奇數次翻轉而 `1` 需要偶數次，因此需要將奇偶分開來討論。  

### 條件 \(c\)： $k$、 $z$、 $m$ 的奇偶限制  

$\sum f_i$ 是 $z$ 個奇數的和，其奇偶性為 $z \bmod 2$。 $\sum g_i$ 是偶數之和，一定為偶數。故：  

$$mk = \sum f_i + \sum g_i \equiv z + 0 \equiv z \pmod{2}$$  

$$\boxed{mk \equiv z \pmod{2}}$$  

- $m$ 為奇數 $\Longrightarrow$ $mk$ 的奇偶性等於 $k$ 的奇偶性 $\Longrightarrow$ 需要 $k \equiv z \pmod{2}$。  
- $m$ 為偶數 $\Longrightarrow$ $mk$ 為偶數 $\Longrightarrow$ 需要 $z \equiv 0 \pmod{2}$。  

整理成表格：  

| $k \equiv z \pmod{2}$ | $z \equiv 0 \pmod{2}$ | 可用的 $m$ |
|:---:|:---:|:---|
| ✓ | ✓ | 奇數與偶數皆可 |
| ✓ | ✗ | 僅奇數 |
| ✗ | ✓ | 僅偶數 |
| ✗ | ✗ | 無解（回傳 $-1$） |


## 4. 合併公式  

綜合以上，分奇偶兩條路：  

**$m$ 為奇數：** 取兩個下界的較大值，再向上調整為奇數。  

```math
m_{\text{odd}} = \max\left(\left\lceil \frac{z}{k} \right\rceil, \left\lceil \frac{w}{b} \right\rceil\right)
```

若 $m_{\text{odd}}$ 為偶數則加 1。有效條件： $k \equiv z \pmod{2}$。  

**$m$ 為偶數：** 取兩個下界的較大值，再向上調整為偶數。  

```math
m_{\text{even}} = \max\left(\left\lceil \frac{z}{k} \right\rceil, \left\lceil \frac{z}{b} \right\rceil\right)
```

若 $m_{\text{even}}$ 為奇數則加 1。有效條件： $z \equiv 0 \pmod{2}$。  

```math
\boxed{ \text{答案} = \min\Big( \bigl\{m_{\text{odd}} \mid k \equiv z \pmod{2}\bigr\} \cup \bigl\{m_{\text{even}} \mid z \equiv 0 \pmod{2}\bigr\} \Big) }
```

若集合為空（即 $k \not\equiv z \pmod{2}$ 且 $z$ 為奇數同時成立），則回傳 $-1$。  

## 5. Python 實作  

```python
import math

class Solution:
    def minOperations(self, s: str, k: int) -> int:
        n = len(s)
        z = s.count('0')

        if z == 0:
            return 0
        if n == k:
            return 1 if z == n else -1

        w = n - z # `1` 的數量
        b = n - k # 每次操作中沒有被翻轉的數量
        candidates = []

        # m 為奇數：有效條件 k ≡ z (mod 2)
        if k % 2 == z % 2:
            m_odd = max(math.ceil(z / k), math.ceil(w / b))
            m_odd += ~m_odd & 1  # 若為偶數則加 1
            candidates.append(m_odd)

        # m 為偶數：有效條件 z ≡ 0 (mod 2)
        if z % 2 == 0:
            m_even = max(math.ceil(z / k), math.ceil(z / b))
            m_even += m_even & 1  # 若為奇數則加 1
            candidates.append(m_even)

        return min(candidates) if candidates else -1
```
